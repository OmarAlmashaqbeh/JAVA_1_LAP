## 📌 وصف المستودع

هذا المستودع يحتوي على حلول (Lap Sheets) لمادة **مختبر البرمجة الكينونية 1 (Object-Oriented Programming 1)**  
في الجامعة الهاشمية، وهو مخصص لمساعدة الطلاب على:

- تعلم أساسيات لغة Java
- فهم إدخال وإخراج البيانات باستخدام:
  - Console (Scanner)
  - Dialog Boxes (JOptionPane)
- تطبيق الجمل الشرطية (if / switch)
- استخدام الحلقات (loops)
- كتابة الدوال (Methods)
- التعامل مع المصفوفات (Arrays) بأنواعها
- فهم أساسيات البرمجة الكينونية (OOP)

---

## 🧠 ماذا ستتعلم؟

من خلال هذا المستودع ستتعلم:

- أساسيات لغة Java
- استخدام Scanner و JOptionPane
- الجمل الشرطية والتكرارية
- التعامل مع Arrays (1D & 2D)
- إنشاء واستخدام Methods
- التعامل مع Strings وملفات (Files)
- أساسيات OOP مثل:
  - Classes & Objects
  - Constructors
  - Encapsulation
  - Access Modifiers
  - Static vs Instance
